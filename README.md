# Todo API (FastAPI + PostgreSQL)

API simple de tareas (CRUD) hecha con FastAPI, SQLAlchemy y PostgreSQL.

## Features
- Crear tarea
- Listar tareas
- Obtener tarea por ID
- Actualizar tarea
- Eliminar tarea

## Requisitos
- Python 3.10+
- PostgreSQL
